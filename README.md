# Car-Rental-Website

A web-based car rental management platform developed using Flask and SQLAlchemy.

This project simulates a real-world car rental system, allowing the management of vehicles, customers, and rental bookings.

## Technologies Used

- Python
- Flask
- SQLAlchemy
- HTML
- CSS

## Features

- Vehicle registration and management  
- Customer registration and management  
- Rental booking system  
- Database integration and data persistence  
- Form validation and business logic handling  

## Project Structure

- `app.py` → Main application file  
- `templates/` → HTML pages  
- `static/` → CSS and images  
- `database/` → SQLite database files
